## Local Session (Firebase)
```bash
firebase serve
```

## Deployment
- Install top level dependencies
```bash
npm install
```
- Install module dependencies
```bash
cd functions/
npm install
```
- Deploy to Firebase
```bash
// From root of project
npm run build
firebase deploy
```
